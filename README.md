# 📋 EggoWorld - NFT Selling System

**Project Type:** NFT Membership System with Multi-Level Commission
**Start Date:** 2026-03-09

---

## 🎯 Project Overview

EggoWorld is an NFT-based membership system where users:

1. **Buy NFT Egg** (25 USDT)
2. **Receive:** Egg NFT + Food NFT
3. **Get:** 30-day membership
4. **Direct recommender** gets commission

**Commission Distribution:**

| Generation | Commission | Condition |
|------------|-------------|------------|
| Project (base) | Remaining | After all gens |
| Gen 1 | 20% (5 USDT) | If recommender exists |
| Gen 2 | 10% (2.5 USDT) | If Gen1 has recommender |
| Gen 3 | 10% (2.5 USDT) | If Gen2 has recommender |
| Gen 4 | 10% (2.5 USDT) | If Gen3 has recommender |

**Total Sale:** 25 USDT
**Max Commission:** 12.5 USDT (50%)
**Project Retains:** 12.5-25 USDT (depending on referrer depth)

## 🔄 Flowchart

```
Member buys Egg NFT (25 USDT)
         ↓
Receive Egg NFT + Food NFT
         ↓
Get 30-day membership
         ↓
Identify direct recommender
         ↓
    ┌────┴────┐
    │         │
  Gen1?     Yes → Distribute 20% (5 USDT) to Gen1
    │         │
   No        └─→ Gen2? → Distribute 10% (2.5 USDT) to Gen2
    │                    │
    │                   No → Gen3? → Distribute 10% (2.5 USDT) to Gen3
    │                    │
    │                   No → Gen4? → Distribute 10% (2.5 USDT) to Gen4
    │                    │
    └────────────────────┘
         ↓
    Remaining commission → Project wallet
         ↓
      Update records
```
